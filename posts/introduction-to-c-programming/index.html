Variables

The following 4 information about the variables should definitely be known.

  1. Name of the variable: The variable must be given a name so that it has a memory location so that it can be used when called.
  2. Type of the variable: Variable types that can hold data should be known.
  3. Space the variable will occupy in memory: How much space it will occupy should be determined.
  4. The value of the veriable: It is necessary to know where the variable can be used and where it can not be used.

Simple Example Of Using Variables

#include <stdio.h>

int main(){

        int a = 10;
        int b = 3, c = 4;
        int x;

        printf("a:%d, b:%d, c:%d\n", a, b, c);

        return 0;
}

In the sample code, part of which is given; Firstly, one variable is defined. Type is int(eger) and 10 in it. In the same way, variables b, c are also defined. The variable x is a space until something is put into it.

Here, %d written in quotes with the printf function; take the variable that is put after this quote and that means pu it where you see %d. Sorting is important at this stage. ıf the order is not taken into consideration, different definitions may come to the desired place.

Here is the code printf("a:10, b:3, c:4"); It could have been written as, but here the code becomes completely fixed, in the future, it was necessary to change all the values one by one if it was desired to make changes. When this code is written as it was originally written, if a change is desired, it will be sufficient to change only the values defined for the variables.

When we run the code written so far, we will get a:10, b:3, c:4 output.

We can add the following lines to take this code one step further.

Examining Variables With Different Examples

#include <stdio.h>

int main(){

        float pi = 3.14;
        printf ("%8.4f", pi);

        return 0;
}

In this sample code; One pi variable is defined. It’s type is float and it has 3.14 in it. Afterwards, it was requested to be printed on the screen as output. Here, the expression %8.4f means that the variable will occupy a total of 8 characters and the decimal part will consist of 4 digits.

After this code is run correctly, 3.1400 will be output.
