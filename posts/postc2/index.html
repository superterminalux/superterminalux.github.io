Basic Conditions for Programming

Statements

Explain the given basic orders and commands. For example printf command is a direct command and an order but like this int main() commands can not to be a statement. End of the sentence there has to be ; (semicolon). Finally this is an atomic property.

Blocks

Used for view multiple statements as a general group. A group is shown between of { ... }the 2 curly brackets.

Why It’s necessiarly?: In the some functions, the command lines for to be run the written statements must be shown in their all. Like this cases, the desired blocks are run for the specific conditions or functions.

Structured Programming

If a coding language has these 3 conditions (like C Language), it can be define as a structured programming language.

  • Can block be put under condition? -> (if-else staitements)
  • Can block work again like a loop? -> (for-while)
  • Can variables be define in the block and can the block be run when variables changed? -> (functions)

Command Comment

  • The part with starts /* is the enter of a multiline comment. This multiline comment is finished with */ also.

    • (If a person wants to think like a machine; this section is can skipped but if a person thinks like a programmer; the informations and comments in the command line(s) may be useful for to the programmer.)

  • With the #include <stdio.h> the standard input-output file is included and this is necessairly for the work of the program.

    • (The #include <conio.h> is required for the some compiliers such as Windows and dev c++. (Required for the using the getch() command.))

  • A basic function is can defined with int main(). For the advanced users, this function type may different and they can write functions like other function types.

  • A block is start is defined for the int main() using { (curly barcket).

  • Line Comment is used starting with // (Two Slash). The machine will not read and run this command line.

  • Variable a is defined to 10 with using int a=10;. That’s mean is there is a variable defined, it occupies a place in the as an integer and 10 be placed in it. Now there is a variable defined in meory and it contains 10. It will be sufficent for the entered/basic level.

  • A new block is opened using with { in the main block. New systems will be defined in it.

  • With // a new comment line is created again. The machine will not read and work here.

  • With int a=5;, variable defined a is created and the value 5 is asiigned into it.

    • In here this question will come to mind; The variable is defined out of the block as int a=10 but in the block it’s defined as int a=5; if anyone want to get a output, which command line will be used for output?

A useful rule to know: if output is wanted, the value in that block will be used. The defined variable will be used for that block only. This name is Scope of Variable.

  • With used }, the block closed.
