Programs

What Is The Program?

The meaning of the program is the executable text specific to hardware of the devices. 1 and 0 codes are run as much as the processor of the device acceptes.

If we examine the working of the program with an example : When a program with .exe extension or a program is run in a diffrent operating systems, certain commands are sent to the processor simultaneously with the execution of the binary codes of this program. These commands try to operate by occupying certain places in memory. (This steps the same for systems like tablets, computers, smartphones, etc.)

The general expression that expresses all these stages is called programming.

Importance Of Veriables In Programming

An important concept to know about programming is veriables. They’re structures that can hold different information in veriables. For example; when a number 5 is put into the program, this number 5 is not veriable and does not change, the number is 5 each time the program is run. However, considering a system in which different values will be placed every time it is run, (such as a menu that asks the user to enter the name&surname when the program is opened) these values to be entered into the program are veriables.

In short, veriables are memory units that can receive new or diffrent information and change şn each operation.

Other Basics

Basic Hardwares

Inputs And Outputs

There are many inputs and outputs that different from system to system, software to software. Keyboard, camera, mouse, etc. can be shown as input. Monitor, network, printer, etc. can be shown as output.

The standard accepted input is keyboard and the standard accepted output is monitor.

RAM : Random Access Memory

Primary memory is the concept of randomly accessible memory that we call memory. Veriable values needed in programming, values to be processed are kept in here.

Disc

Usually harddisk, ssd is meant. In computers and other systems; there is a disk called secondary memory and information is stored in it.

CPU : Central Process Unit

It is the unit that provides the central transaction management, which we call the processor. The written codes and the informations and veriables kept in RAM are exrcuted here.

If we give a collective example of what has benn mentoned so far; Let’s calculate the sum of three different number. -For the calculation, the desired values are entered from the keyboard. In here the input is keyboard. -These entered values are kept in RAM. -Numerical calculations are done in the CPU -As a result of these actions, the desired output is obtained with the monitor. In here the output is monitor.

Basic Software Elements

OS : Operating System

Everything works on this system, the programs written work here. (If an operating system will not be written, we will proceed by assuming that no operating system has been written.)

Every program needs an operating system specific functions need to be called in order to run properly.

Source Code

People can read, understand and write this codes.

Machine Code

The machine understands this code. These codes are actually consist of signals consisting of 1s and 0s. Even if the operating system is the same, the 1 and 0 signals be run by different processors, the speed of these signals and the reactions to be given may differ.

Compilers

It’s the unit that converts the source code to machine code and signals.
